# ShareHouse-LW-2
Lite-weight version of "ShareHouse" engine for group 2.

=== Using ===

IDE : Visual Studio 2019
CPPVersion : C++ 17
GraphicsAPI : D3D11

=== Third Party === 

● spdlog 
https://github.com/gabime/spdlog

=== Coding Rules ===

Class에는 SH를 접두사로 붙여 네이밍
Struct에는 F를 접두사로 붙여 네이밍
Enum에는 E를 접두사로 붙여 네이밍
